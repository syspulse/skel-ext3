# Experimental Detectors

This is a repository for experimental detectors.

Some of them are generated by AI

Detectors are based on [Sentinel Framework Scala SDK](https://github.com/haas-labs/ext-sentinel)

- [DetectorAaveGov](sentry-ext/src/main/scala/io/syspulse/ext/sentinel/DetectorAaveGov.scala) (generated 99% by Claude)


## Running

```
$SKEL_HOME/classpath.sh ..
```

Every detector can be run from their directory with

```
../sentinel.sh --conf=conf/detector-test-1.conf 
```

Any parameters in config can be overrided (e.g. `--chain=base`)
